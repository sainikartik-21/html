/* Color and Text */
color: Specifies the text color of an element.
Example: color: red;

background-color: Sets the background color of an element.
Example: background-color: lightblue;

/* Borders */
border: Shorthand for defining the border width, style, and color.
Example: border: 2px solid black;

border-radius: Defines the rounded corners of an element.
Example: border-radius: 10px;

/* Dimensions */
width: Specifies the width of an element.
Example: width: 400px;

height: Specifies the height of an element.
Example: height: 200px;

/* Background */
background-image: Sets an image as the background.
Example: background-image: url('image.jpg');

background-repeat: Defines how the background image is repeated.
Example: background-repeat: no-repeat;

background-size: Specifies the size of the background image.
Example: background-size: cover;

/* Gradient Background */
background: Allows setting a linear or radial gradient as a background.
Example: background: linear-gradient(to right, #4b6cb7, #182848);
